Describe the biggest difference between .forEach & .map.

The forEach() method executes a provided function once for each array element while the map() method creates a brand new array

What is the difference between a function and a method?

A function is an object and the method executes actions for that function or object.

What is closure?

A closure allows functions in different scopes to access each other

Describe the four rules of the 'this' keyword.

Global/Window scope allows the "this" keyword to travel out or globally.
Implicit Binding allows functions to be called using the dot notation but only when the object is before the dot.
New binding allows constructor functions to return objects to its original creator.
Explicit binding allows functions to use call() or apply() methods only when "this" is used
